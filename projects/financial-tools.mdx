# Financial Tools

This category features sophisticated financial management applications designed for personal and business use. These tools combine modern web technologies with intuitive user interfaces to provide powerful financial insights and management capabilities.

## Featured Projects

### [Money Tracker](./individual-projects/money-tracker.mdx)
A comprehensive expense tracking web application that provides real-time financial monitoring with advanced data visualization capabilities.

**Key Features:**
- Real-time expense tracking with immediate visual feedback
- Advanced data visualization using Chart.js
- Local storage for privacy-first architecture
- Multi-currency support (20+ currencies)
- Responsive design for all devices
- Import/export functionality (JSON/CSV)

**Technology Stack:**
- Frontend: Vanilla JavaScript (ES6+), HTML5, CSS3
- Visualization: Chart.js 4.x
- Storage: Browser LocalStorage
- Icons: Font Awesome 6.4
