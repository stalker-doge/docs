## Featured Projects

### [FF14 Gillionaire](./individual-projects/ff14-gillionaire.mdx)
A sophisticated dashboard for Final Fantasy XIV that analyzes item profitability by comparing vendor prices with real-time marketboard data.

**Key Features:**
- Real-time market data from Universalis API
- Hourly price updates across European data centers
- Beautiful pastel pink UI optimized for long viewing sessions
- Advanced data visualization with Recharts
- Item details with listing history and price trends
- Responsive design for desktop and mobile

**Technology Stack:**
- Backend: Node.js, Express.js, MongoDB
- Frontend: React 18, Material-UI
- APIs: XIVAPI, Universalis API
- Visualization: Recharts
- Deployment: Docker, PM2
